\# Revolut Clone 💳



A full-stack fintech prototype featuring a Flutter frontend and a FastAPI (Python) backend.



\## 🚀 Features

\*   \*\*Live Balance:\*\* Sending money in the app updates the balance on the Python server.

\*   \*\*Dynamic Transactions:\*\* Every transfer to \*\*Dilawar\*\* is recorded and shown in the history.

\*   \*\*Charts:\*\* Visualized spending data with animated UI components.

\*   \*\*Dual Mode:\*\* Supports both Light and Dark mode themes.



\## 🛠️ Tech Stack

\*   \*\*Frontend:\*\* Flutter (Dart)

\*   \*\*Backend:\*\* FastAPI (Python)

\*   \*\*Communication:\*\* REST API (JSON)



\## 🏃 How to Run

1\.  \*\*Backend:\*\* 

&#x20;   \*   `cd backend`

&#x20;   \*   `.\\venv\\Scripts\\activate`

&#x20;   \*   `uvicorn main:app --reload --host 0.0.0.0`

2\.  \*\*Frontend:\*\*

&#x20;   \*   `cd frontend`

&#x20;   \*   `flutter run`
